About This Tool

  • Runs entirely in your browser — no data is sent to any server.
  • Applies a 5-step EAR classification pipeline: Subject to EAR → ECCN mapping → End-use/user → Country controls → Licence indication.
  • All outputs are indicative and non-binding. Confidence is reduced where data files contain [TODO] thresholds.
  • Data files are snapshots — run scripts/refresh_rules.py or trigger refresh-rules.yml to update.
  • ECCN thresholds for advanced computing (3A090, 4A090) and AI model weights are marked [TODO] and must be populated manually from the current CCL.
  • Covers EAR only. OFAC sanctions and ITAR require separate analysis.
